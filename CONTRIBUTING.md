# Contributing to CA Office Suite

Thank you for your interest in contributing to the CA Office Suite!

## Getting Started

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes.
4. Run tests and linting.
5. Submit a Pull Request.

## Code Style

- Follow the existing code style.
- Ensure all tests pass.
- Update documentation as needed.

## Reporting Issues

Please use the [GitHub Issues](https://github.com/Srivari-Hema-SSPL-2026/ca-office-suite/issues) tracker to report bugs or request features.
